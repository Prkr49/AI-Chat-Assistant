# WhatsApp Auto Reply Chatbot 🤖

A Python automation bot that reads WhatsApp Web messages and replies automatically using OpenAI.

## Features
- Reads WhatsApp chat
- Detects last sender
- Generates AI reply
- Sends reply automatically

## Tech Stack
- Python
- PyAutoGUI
- Pyperclip
- OpenAI API

## How to Run

pip install -r requirements.txt
python main.py
